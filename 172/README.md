# 离轴数字全息图重建 (Off-axis Digital Holography Reconstruction)

## 项目简介

本项目实现了离轴数字全息图的数值重建，通过傅里叶变换方法滤除零级像和共轭像，恢复物体的复振幅（幅度和相位）。

## 原理概述

### 离轴全息术

离轴全息术通过让参考光与物光成一定角度入射，使得全息图的傅里叶频谱中三个分量（零级、+1级、-1级）在空间上分离：

```
全息图强度: I = |O + R|² = |O|² + |R|² + O*R* + O*R
```

其中:
- `|O|² + |R|²`: 零级项（直流分量）
- `O*R*`: -1级项（共轭像）
- `O*R`: +1级项（物波信息）

### 重建流程

```
全息图 → 2D-FFT → 频谱 → 提取+1级衍射斑 → 2D-IFFT → 复振幅 → 幅度/相位
```

## 核心代码解释

### 1. 全息图生成

```python
object_wave = amplitude * np.exp(1j * phase)  # 物波
ref_wave = np.exp(1j * k * X * sin(theta))     # 倾斜参考光
hologram = np.abs(object_wave + ref_wave)**2   # 干涉条纹
```

### 2. 傅里叶变换与频谱滤波

```python
spectrum = fftshift(fft2(hologram))           # 二维傅里叶变换
filtered_spectrum = spectrum * mask           # 提取+1级
```

### 3. 波前重建

```python
reconstructed_wave = ifft2(ifftshift(filtered_spectrum))
amplitude = np.abs(reconstructed_wave)        # 幅度
phase = np.angle(reconstructed_wave)          # 相位
```

### 4. 相位处理

```python
phase_unwrapped = np.unwrap(phase)            # 相位解包裹
phase_corrected = remove_linear_phase(phase_unwrapped)  # 去除倾斜
```

## 改进技术详解

### 1. 汉宁窗减少频谱泄漏

**问题**: 直接对图像做FFT会因为边界不连续产生频谱泄漏，导致衍射级之间相互混叠。

**解决方案**: 在FFT前乘以汉宁窗函数，使图像边界平滑过渡到零。

```python
window = np.outer(hann(rows), hann(cols))  # 二维汉宁窗
holo_windowed = hologram * window         # 加窗处理
```

### 2. 自动峰值检测与聚类

**问题**: 手动选择ROI主观性强，难以重复。

**解决方案**:
1. **局部最大值检测**: 使用`maximum_filter`寻找频谱峰值
2. **K-means聚类**: 将峰值聚为3类（零级、+1级、-1级）
3. **自动分类**: 根据距离中心的远近和水平位置识别各衍射级

### 3. 余弦窗滤波减少吉布斯效应

**问题**: 矩形窗的硬截断会产生吉布斯振荡，导致重建图像出现振铃伪影。

**解决方案**: 使用升余弦窗（Tukey窗），使滤波边界平滑过渡：

```python
window[dist <= radius - edge_width] = 1  # 通带内为1
window[edge_region] = 0.5 * (1 + cos(π * transition))  # 过渡带余弦下降
```

### 4. 频谱中心化

将+1级衍射斑平移到频谱中心，消除重建相位中的线性倾斜分量，简化后续相位校正。

## 数学背景

离轴全息的优势在于可以通过简单的空间滤波分离物波信息，避免了同轴全息中的孪生像问题。

**频谱位置**:
- 零级: 频谱中心 (0,0)
- +1级: (f_x = sin(θ)/λ, 0)
- -1级: (f_x = -sin(θ)/λ, 0)

## 活细胞DH定量分析

**相位→高度转换公式**:
```
h = (φ * λ) / (2π * Δn)
```
其中:
- φ: 相位差 (rad)
- λ: 波长 (m)
- Δn: 细胞与培养基折射率差 (典型值: 0.03~0.07)

**典型活细胞参数**:
- 细胞高度: 1~10 μm
- 相位差: ~0.5~5 rad
- 折射率: 1.36~1.40 (细胞质)
