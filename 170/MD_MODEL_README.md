# Maugis-Dugdale (MD) 接触力学模型说明

## 问题背景

在原子力显微镜（AFM）的力-距离曲线拟合中，传统的DMT和JKR模型在某些情况下会遇到数值发散问题，特别是在**小接触半径（大曲率半径）**时：

- **DMT模型**：适用于硬材料、小粘附力、小半径的情况，但在过渡区域拟合不佳
- **JKR模型**：适用于软材料、大粘附力、大半径的情况，但在小半径时可能不稳定

## Maugis-Dugdale 模型的优势

MD模型通过引入Tabor参数（λ）实现了DMT和JKR之间的平滑过渡：

### Tabor参数 (λ)
```
λ = σ₀ * (9R / (πE*z₀²))^(1/3)
```
- σ₀: 理论粘附应力
- R: 针尖半径
- E: 杨氏模量
- z₀: 原子间平衡距离（默认0.3nm）

### 模型过渡行为
- **λ << 1** → 趋近于DMT模型
- **λ ≈ 1** → 过渡区域（MD模型最有价值的区域）
- **λ >> 1** → 趋近于JKR模型

## 数值稳定性改进

1. **边界值限制**：λ被限制在[0.01, 10.0]范围内，防止极端值导致的发散

2. **回退机制**：当数值求解失败时，自动使用DMT和JKR的加权平均作为备用方案：
   ```python
   w = 1 / (1 + λ)
   F = w * F_dmt + (1 - w) * F_jkr
   ```

3. **安全的数值求解**：使用brentq方法并设置合理的搜索区间，确保收敛

4. **小压痕处理**：对δ ≤ 0的情况，使用指数衰减模型平滑处理

## 使用示例

```python
from afm_fitting import AFMCurveFitter

# 创建拟合器（使用MD模型推荐的参数）
fitter = AFMCurveFitter(
    radius=5e-9,    # 小半径时MD模型特别有用
    nu=0.5, 
    z0=0.3e-9       # 原子间距离，可根据材料调整
)

# 使用MD模型拟合（默认）
fitter.fit(delta, force, model='MD')

# 打印结果
fitter.print_results()

# 对比所有模型
for model in ['DMT', 'JKR', 'MD']:
    fitter.fit(delta, force, model=model)
    r2 = fitter.calculate_goodness_of_fit(delta, force)
    print(f"{model}: R² = {r2:.6f}")
```

## 何时使用MD模型

**推荐使用MD模型的情况**：
1. 针尖半径较小（< 20 nm）
2. 粘附力处于中等水平
3. DMT和JKR拟合都不理想（R²较低）
4. 需要在不同实验条件下保持拟合一致性

**模型选择参考**：
| 材料硬度 | 粘附力 | 推荐模型 |
|---------|--------|---------|
| 很硬 | 很小 | DMT |
| 中等 | 中等 | MD |
| 很软 | 很大 | JKR |

## 参数说明

| 参数 | 符号 | 默认值 | 说明 |
|------|------|--------|------|
| radius | R | 10e-9 m | 针尖半径 |
| nu | ν | 0.5 | 泊松比 |
| z0 | z₀ | 0.3e-9 m | 原子间平衡距离 |
| E | E | - | 杨氏模量（拟合输出） |
| F_ad | F_ad | - | 粘附力（拟合输出） |
