# 多轴疲劳评估说明

## 概述

本工具实现了多种多轴疲劳评估方法，适用于拉扭组合等复杂应力状态下的疲劳寿命预测。

## 核心类说明

### 1. StressTensor - 应力张量类
用于表示三维应力状态，支持主应力、von Mises等效应力计算。

```python
# 创建应力张量
stress = StressTensor(sigma_xx=300, sigma_yy=0, sigma_zz=0,
                      tau_xy=150, tau_yz=0, tau_xz=0)

# 计算主应力
principal = stress.principal_stresses()

# 计算von Mises等效应力
von_mises = stress.von_mises()

# 获取某平面上的应力 (θ为极角，φ为方位角)
sigma_n, tau = stress.stress_on_plane(theta=np.pi/4, phi=0)
```

### 2. LoadHistory - 载荷历史类
用于构建和管理循环载荷历史，特别支持拉扭组合载荷。

```python
# 创建载荷历史对象
load = LoadHistory()

# 生成循环拉扭组合载荷
load.generate_cyclic_load(
    n_cycles=3,           # 循环数
    n_points_per_cycle=36, # 每周期点数
    sigma_axial_amp=300,  # 轴向应力幅值
    sigma_axial_mean=0,   # 轴向平均应力
    tau_torsion_amp=200,  # 扭转剪应力幅值
    tau_torsion_mean=0,   # 扭转平均剪应力
    phase_shift=0          # 相位差 (0=同相, π/2=90°异相)
)
```

### 3. MultiaxialFatigue - 多轴疲劳评估类
实现了多种多轴疲劳准则。

## 多轴疲劳准则

### 1. Findley准则
**临界平面法**，考虑剪应力幅和最大法向应力的综合影响：

```
F = τ_amp + k × max(σ_n_max, 0)
```

- **k值**: 通常0.5~1.0，钢材约0.8
- **特点**: 物理意义明确，工程应用广泛

```python
result = fitter.evaluate_tension_torsion(
    sigma_amp=300,
    tau_amp=200,
    method='findley'
)

print(f"损伤参数: {result['max_damage']:.2f} MPa")
print(f"临界平面: θ={result['critical_theta']:.1f}°")
print(f"预测寿命: {result['life']:.2e}")
```

### 2. SWS临界平面法 (Smith-Watson-Topper)
**基于能量的临界平面法**:

```
SWS = √(σ_n_amp × max(σ_n_max, 0))
```

- **特点**: 对非比例加载敏感，适合复杂加载路径

```python
result = fitter.evaluate_tension_torsion(
    sigma_amp=300,
    tau_amp=200,
    method='sws'
)
```

### 3. Brown-Miller准则
考虑剪应变和法向应变的组合:

```
D = γ_amp + k × ε_n_max
```

适用于低周疲劳分析。

## 完整使用示例

### 示例1: 基本拉扭组合评估

```python
from sn_curve_fit import SNCurveFitter

# 1. 创建拟合并拟合S-N曲线

fitter.fit()

# 2. 多轴疲劳评估
result = fitter.evaluate_tension_torsion(
    sigma_amp=300,   # 轴向应力幅
    tau_amp=200,     # 扭转剪应力幅
    sigma_mean=50,   # 轴向平均应力
    tau_mean=0,      # 扭转平均应力
    phase_shift=0,   # 同相加载
    method='findley' # 选择方法
)

print(f"Findley参数: {result['max_damage']:.2f}")
print(f"预测寿命: {result['life']:.2e}")
```

## 方法选择建议

| 方法 | 适用场景 | 优点 | 注意事项 |
|-----|---------|------|---------|
| **Findley** | 比例/非比例加载 | 物理直观，工程常用 | 需校准k值 |
| **SWS** | 非比例加载，应变寿命 | 能量准则，理论完善 | 计算稍复杂 |
| **Brown-Miller** | 低周疲劳 | 考虑法向和剪切耦合 | 适合应变控制 |
| **von Mises** | 快速估算 | 计算简单 | 偏危险，非比例加载误差大 |

## 临界平面解释

程序输出的临界平面角度:
- **θ (theta)**: 极角，与z轴（轴向）的夹角 0°~180°
- **φ (phi)**: 方位角，在xy平面内的旋转 0°~360°

**典型情况**:
- 纯拉伸: θ=45°，τ最大平面
- 纯扭转: θ=45°~90°，取决于准则
- 拉扭组合: 取决于应力比

## 参数校准

Findley的k值建议根据试验校准:

```python
# 根据试验数据调整k值
fitter.multiaxial.k_f = 0.75  # 自定义k值

# 重新评估
result = fitter.evaluate_tension_torsion(300, 200)
```

## 注意事项

1. **S-N曲线拟合**: 多轴疲劳评估必须先进行单轴S-N曲线拟合
2. **材料参数**: 确保抗拉强度σ_uts等参数正确设置
3. **载荷点数**: 建议每周期至少36个点以保证精度
4. **计算时间**: 临界平面搜索涉及双重循环（θ×φ），计算量较大
5. **非比例加载**: SWS和Findley比von Mises更适合非比例加载

## 常见问题

**Q: 为什么von Mises预测寿命比临界平面法长？**
A: von Mises不考虑加载路径和附加硬化效应，通常偏于危险，临界平面法更准确。

**Q: 临界平面角度不是45度？**
A: 当有平均应力或非比例加载时，临界平面会偏离最大剪应力平面。

**Q: 压应力时如何处理？**
A: Findley等准则自动忽略压的法向应力项(max(σ_n,0))，符合疲劳物理机制。
