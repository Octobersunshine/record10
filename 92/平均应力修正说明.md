# 平均应力修正方法说明

## 问题背景

原始的Goodman修正公式在处理压应力（σ_m < 0）时存在过度修正问题，导致预测寿命与试验数据偏离。本工具实现了5种修正方法，特别针对压应力问题进行了优化。

## 修正方法对比

### 1. Goodman修正 (经典方法)
```
σ_a_eq = σ_a / (1 - σ_m / σ_uts)  (σ_m ≥ 0)
σ_a_eq = σ_a                         (σ_m < 0)
```
- **问题**: 压应力时完全不修正，预测寿命偏保守（过短）
- **适用**: 仅拉平均应力情况

### 2. Gerber修正
```
σ_a_eq = σ_a / (1 - (σ_m / σ_uts)²)  (σ_m ≥ 0)
σ_a_eq = σ_a                            (σ_m < 0)
```
- **特点**: 抛物线形式，对延性材料拟合较好
- **问题**: 压应力时同样不修正

### 3. Walker修正 (推荐默认)
```
σ_max = σ_a + σ_m
σ_a_eq = σ_a * (2σ_max / (σ_a + σ_max))^γ
```
- **γ = 0.5 (默认)**
- **优点**: 连续函数，拉压应力都有合理修正
- **推荐**: 处理压应力的最佳选择

### 4. SWT (Smith-Watson-Topper)
```
σ_max = max(σ_a + σ_m, 0)
σ_a_eq = √(σ_a * σ_max)
```
- **基于能量方法**
- **优点**: 物理意义明确

### 5. 改进Goodman修正
```
σ_a_eq = σ_a / (1 - σ_m / σ_uts)  (σ_m ≥ 0)
σ_a_eq = σ_a / (1 + k * |σ_m| / σ_uts)  (σ_m < 0)
```
- **k = 0.3**
- **优点**: 对压应力进行温和修正，与试验数据更符合

## 推荐使用方法

```python
from sn_curve_fit import SNCurveFitter

fitter = SNCurveFitter(sigma_uts=1000, sigma_yield=800, walker_gamma=0.5)

# 方法1: 使用默认Walker修正（推荐）
life = fitter.predict_life(sigma_a=300, sigma_m=-200, method='walker')

# 方法2: 使用改进Goodman
life = fitter.predict_life(sigma_a=300, sigma_m=-200, method='goodman_improved')

# 对比所有方法
fitter.compare_correction_methods(sigma_a=300)

# 绘制对比图
fitter.plot_correction_comparison(sigma_a=300, save_path='comparison.png')
```

## 参数调整建议

1. **Walker方法的γ参数**:
   - 钢材: 0.4 ~ 0.6
   - 铝合金: 0.5 ~ 0.7
   - 根据试验数据校准最佳值

2. **改进Goodman的k参数**:
   - 默认0.3，根据材料调整
   - 范围0.2 ~ 0.5

## 注意事项

1. 确保σ_uts（抗拉强度）必须正确设置
2. 压应力情况下避免使用原始Goodman
3. Walker方法为默认推荐，对大多数材料适用
4. 如有试验数据，建议先用几种方法对比后选择
