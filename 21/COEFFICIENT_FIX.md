# 数值积分实现说明

## 概述

本项目实现了两种高精度数值积分方法：
1. **牛顿-柯特斯积分** - 支持n=1到n=6阶公式
2. **龙贝格积分** - 基于理查德森外推的自适应高精度方法

---

## 一、牛顿-柯特斯积分

### 1.1 核心实现

基于伯努利数和拉格朗日插值多项式推导：

**伯努利数计算** (递推公式):
```
B₀ = 1
对于 m ≥ 1: Σ(k=0到m) [C(m+1, k) * Bk] = 0
```

**拉格朗日基函数积分**:
```
Cj = ∫₀ⁿ Lj(t) dt  其中 Lj(t) = Π(m≠j) (t - m) / (j - m)
```

### 1.2 标准牛顿-柯特斯系数表

| n | 公式名称 | 系数 | 权重 |
|---|---------|------|------|
| 1 | 梯形法则 | 1/2 | [1, 1] |
| 2 | 辛普森法则 | 1/3 | [1, 4, 1] |
| 3 | 辛普森3/8法则 | 3/8 | [1, 3, 3, 1] |
| 4 | 布尔法则 | 2/45 | [7, 32, 12, 32, 7] |
| 5 | Milne法则 | 5/288 | [19, 75, 50, 50, 75, 19] |
| 6 | Weddle法则 | 1/140 | [41, 216, 27, 272, 27, 216, 41] |

### 1.3 误差分析

截断误差通过伯努利数表示：
```
E = - (h^(n+2) * (b-a) * Bn+2) / ((n+2)!) * f^(n+2)(ξ)
```

偶数n精度阶数为n+1，奇数n为n。

## 二、龙贝格积分 (Richardson外推)

### 2.1 核心算法

龙贝格积分基于梯形法则递推，通过理查德森外推加速收敛。

**1. 梯形法则递推**:
```
R[i][0] = hᵢ * Σf(xⱼ)  其中 hᵢ = (b-a)/2ⁱ
```

**2. 理查德森外推**:
```
R[i][j] = (4ʲ * R[i][j-1] - R[i-1][j-1]) / (4ʲ - 1)
```

### 2.2 龙贝格积分表结构

```
R[0][0]
R[1][0]  R[1][1]
R[2][0]  R[2][1]  R[2][2]
R[3][0]  R[3][1]  R[3][2]  R[3][3]
...
```

其中R[i][i]是第i阶外推结果，精度为O(h^(2i+2))。

### 2.3 自适应收敛

通过检查对角线元素的变化：
```
|R[i][i] - R[i-1][i-1]| < Tolerance
```

达到精度要求时自动停止迭代。

## 三、API接口说明

### 3.1 龙贝格积分请求

```json
{
  "function": "x^2",
  "a": 0,
  "b": 1,
  "method": "romberg",
  "tolerance": 1e-10,
  "max_levels": 10
}
```

### 3.2 牛顿-柯特斯积分请求

```json
{
  "function": "x^2",
  "a": 0,
  "b": 1,
  "method": "newton-cotes",
  "n": 2,
  "N": 10
}
```

### 3.3 响应格式

```json
{
  "result": 0.333333333333,
  "success": true,
  "method": "romberg",
  "table": [[0.5], [0.375, 0.333333333333], ...]
}
```

## 四、验证方法

### 4.1 系数验证
通过数值积分拉格朗日基函数计算的权重与标准权重对比。

### 4.2 积分精度验证
测试用例：
- f(x) = x² 在 [0,1] 积分 = 1/3
- f(x) = x³ 在 [0,1] 积分 = 1/4
- f(x) = sin(x) 在 [0,π] 积分 = 2
- f(x) = exp(x) 在 [0,1] 积分 = e - 1

### 4.3 运行验证程序

```bash
# 完整验证（伯努利数、系数、龙贝格收敛）
go run verify_coefficients.go

# 启动服务（启动时自动显示系数验证）
go run main.go
```

## 五、方法对比

| 特性 | 龙贝格积分 | 牛顿-柯特斯(n=2) | 牛顿-柯特斯(n=4) |
|------|-----------|-----------------|-----------------|
| 精度阶数 | O(h^(2k)) 自适应 | O(h⁴) | O(h⁶) |
| 收敛速度 | 极快 | 中等 | 快 |
| 参数设置 | 精度容差 | 阶数n、子区间N | 阶数n、子区间N |
| 适用场景 | 高精度要求 | 一般精度 | 平滑函数高精度 |
| 函数求值次数 | 少（自适应） | 固定 | 固定 |

**推荐**：大多数情况下使用龙贝格积分，可自动达到极高精度。
