# Peng-Robinson状态方程拟合工具（稳定版）

## 功能说明

本工具用于根据实验PVT（压力-体积-温度）数据拟合Peng-Robinson状态方程的参数。**特别优化了临界点附近的数值稳定性，采用Heidemann-Khalil临界点条件和阻尼牛顿法确保求解稳定。**

## 主要改进

### 1. Heidemann-Khalil临界点条件
在拟合目标函数中加入临界点稳定性约束，确保拟合得到的参数满足：
- 临界点处一阶导数为零：$\left(\frac{\partial P}{\partial V}\right)_{T_c} = 0$
- 临界点处二阶导数为零：$\left(\frac{\partial^2 P}{\partial V^2}\right)_{T_c} = 0$

### 2. 阻尼牛顿法求解体积根
- 自适应阻尼因子防止迭代发散
- 最大步长限制避免数值振荡
- 最优解回溯机制确保收敛
- 物理边界约束（$V > b$）

### 3. 同伦延拓法（可选）
对于特别困难的求解情况，提供同伦延拓法逐步逼近目标压力。

### 4. 数值稳定性保护
- 安全分母处理，防止除零
- 变量钳位确保物理合理性
- NaN/Inf 检测和异常处理
- 异常捕获机制

## Peng-Robinson状态方程

$$P = \frac{RT}{V - b} - \frac{a\alpha}{V(V + b) + b(V - b)}$$

其中：
- $a = 0.457235 \cdot \frac{R^2 T_c^2}{P_c}$
- $b = 0.077796 \cdot \frac{R T_c}{P_c}$
- $\alpha = \left[1 + \kappa(1 - \sqrt{T_r})\right]^2$
- $\kappa = 0.37464 + 1.54226\omega - 0.26992\omega^2$
- $T_r = T / T_c$

## 使用方法

### 基本使用示例

```python
from pr_eos_fit import PengRobinsonEOS
import numpy as np

# 准备实验数据
V_data = np.array([0.001, 0.002, 0.003, ...])  # 体积，单位：m³/mol
P_data = np.array([1e6, 5e5, 3e5, ...])        # 压力，单位：Pa
T_data = np.array([300, 300, 300, ...])        # 温度，单位：K

# 创建拟合对象
pr_eos = PengRobinsonEOS(R=8.314, epsilon=1e-12)

# 执行拟合（可调节权重平衡拟合精度和稳定性）
params = pr_eos.fit(
    V_data, P_data, T_data,
    weight_critical=500.0,      # 临界点条件权重
    weight_pressure=1.0,         # 压力拟合权重
    weight_consistency=100.0     # 参数一致性权重
)

# 打印结果
pr_eos.print_results()

# 使用拟合参数预测压力
P_pred = pr_eos.predict_pressure(V_data, T_data)

# 使用拟合参数预测体积（阻尼牛顿法）
V_liquid = pr_eos.predict_volume(P_data, T_data, phase='liquid')
V_vapor = pr_eos.predict_volume(P_data, T_data, phase='vapor')

# 使用同伦延拓法（对于困难情况）
V_stable = pr_eos.predict_volume(P_data, T_data, phase='vapor', use_homotopy=True)
```

### 返回参数说明

拟合成功后，`params` 字典包含以下参数：

| 参数 | 说明 | 单位 |
|------|------|------|
| `a` | Peng-Robinson方程参数a | Pa·m⁶/mol² |
| `b` | Peng-Robinson方程参数b | m³/mol |
| `Tc` | 临界温度 | K |
| `Pc` | 临界压力（理论值） | Pa |
| `Pc_calc` | 临界压力（计算值） | Pa |
| `omega` | 偏心因子 | 无量纲 |
| `kappa` | α函数中的κ参数 | 无量纲 |
| `dPdV_critical` | 临界点处一阶导数 | Pa·mol/m³ |
| `d2PdV2_critical` | 临界点处二阶导数 | Pa·mol²/m⁶ |
| `success` | 拟合是否成功 | 布尔值 |
| `message` | 优化器返回的消息 | 字符串 |
| `objective_value` | 目标函数最终值 | 无量纲 |

### 权重参数调整

根据数据情况调整权重参数：

```python
# 情况1：数据质量好，包含临界点附近数据
params = pr_eos.fit(
    V_data, P_data, T_data,
    weight_critical=100.0,   # 降低临界点约束权重
    weight_pressure=1.0,
    weight_consistency=50.0
)

# 情况2：数据远离临界点，需要强物理约束
params = pr_eos.fit(
    V_data, P_data, T_data,
    weight_critical=1000.0,  # 提高临界点约束权重
    weight_pressure=1.0,
    weight_consistency=200.0
)

# 情况3：优先保证拟合精度
params = pr_eos.fit(
    V_data, P_data, T_data,
    weight_critical=50.0,
    weight_pressure=5.0,     # 提高压力拟合权重
    weight_consistency=50.0
)
```

## 数值稳定性问题排查

### 问题1：拟合不收敛
**可能原因**：
- 初始猜测值不合理
- 数据单位错误
- 数据点分布不合理

**解决方法**：
```python
# 1. 调整初始猜测值
initial_guess = {'a': 0.5, 'b': 8e-5, 'Tc': 300, 'omega': 0.2}
params = pr_eos.fit(V_data, P_data, T_data, initial_guess=initial_guess)

# 2. 增加临界点约束权重
params = pr_eos.fit(
    V_data, P_data, T_data,
    weight_critical=2000.0
)
```

### 问题2：体积求解失败
**可能原因**：
- 压力/温度在临界点附近
- 初始值选择不当

**解决方法**：
```python
# 使用同伦延拓法
V = pr_eos.predict_volume(P_data, T_data, phase='vapor', use_homotopy=True)
```

### 问题3：临界点条件不满足
**检查方法**：
```python
print(f"dP/dV (临界点) = {params['dPdV_critical']:.6e}")
print(f"d²P/dV² (临界点) = {params['d2PdV2_critical']:.6e}")
```

**解决方法**：增加 `weight_critical` 参数值。

## 注意事项

1. **数据单位**：请确保输入数据的单位正确
   - 压力：帕斯卡 (Pa)
   - 体积：立方米每摩尔 (m³/mol)
   - 温度：开尔文 (K)

2. **数据质量**：
   - 建议使用多个温度下的数据点
   - 包含亚临界和超临界数据有助于稳定拟合
   - 避免数据点过于靠近两相区

3. **权重调整**：
   - `weight_critical`: 控制临界点条件的严格程度
   - `weight_pressure`: 控制压力拟合的精度
   - `weight_consistency`: 控制a、b、Tc之间的理论关系

4. **物理边界**：
   - 体积必须大于协体积参数b
   - 温度必须为正值
   - 偏心因子通常在0-1.5之间

## 算法细节

### 目标函数

$$\text{Total Error} = w_P \cdot \text{Pressure Error} + w_C \cdot \text{Critical Error} + w_{cons} \cdot \text{Consistency Error}$$

其中：
- Pressure Error: 压力相对误差平方和
- Critical Error: 归一化的临界点导数平方和
- Consistency Error: 参数理论一致性误差

### 阻尼牛顿法迭代公式

$$V_{k+1} = V_k + \lambda \cdot \frac{P(V_k) - P_{\text{target}}}{\left(\frac{dP}{dV}\right)_{V_k}}$$

其中 $\lambda \in (0, 1]$ 为自适应阻尼因子。
