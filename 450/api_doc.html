GET/api/v1/products获取产品列表
分页获取所有上架产品
参数
参数名类型必填位置默认值说明
pageinteger (原: String)否query-页码 (默认1)
limitinteger (原: String)否query-每页数量 (默认20)
categoryString是query-产品分类ID
sortString否query-排序字段 (price/sales/created)
返回字段
字段类型说明
totalinteger (原: Number)总数量
productsArray产品列表
products.idString产品ID
products.nameString产品名称
products.priceinteger (原: Number)价格(分)
products.stockinteger (原: Number)库存
成功响应示例
{ "total": 100, "products": [ { "id": "p_001", "name": "无线蓝牙耳机", "price": 19900, "stock": 56 } ] }

POST/api/v1/orders创建订单
提交新订单
参数
参数名类型必填位置默认值说明
product_idString是query-产品ID
quantityinteger (原: Number)是query-购买数量
coupon_codeString否query-优惠码
address_idString是query-收货地址ID
返回字段
字段类型说明
order_idString订单ID
total_amountinteger (原: Number)总金额(分)
statusString订单状态
成功响应示例
{ "order_id": "ord_20240101_001", "total_amount": 39800, "status": "pending" }
错误响应示例
{ "error": "product_not_found", "message": "产品不存在" }

DELETE/api/v1/orders/{order_id}取消订单
取消指定订单（仅限待支付状态）
参数
参数名类型必填位置默认值说明
order_idString是query-订单ID (路径参数)
reasonString否query-取消原因
返回字段
字段类型说明
order_idString订单ID
statusString更新后状态
成功响应示例
{ "order_id": "ord_20240101_001", "status": "cancelled" }

GET/api/v1/users获取用户列表
根据筛选条件分页查询用户信息
参数
参数名类型必填位置默认值说明
pageinteger否query-页码，默认1
limitinteger否query-每页数量，默认20
keywordstring否query-搜索关键词
rolestring是query-角色筛选 (admin/user/guest)
返回字段
字段类型说明
totalinteger总记录数
usersarray用户列表
users.idstring用户ID
users.namestring用户名
users.emailstring邮箱地址
成功响应示例
{ "total": 42, "users": [ { "id": "u_001", "name": "Alice", "email": "alice@example.com" } ] }

POST/api/v1/users创建新用户
提交用户信息创建新账号
参数
参数名类型必填位置默认值说明
namestring是query-用户名，2-50个字符
emailstring是query-邮箱地址
passwordstring是query-密码，至少8位
rolestring否query-角色，默认user
返回字段
字段类型说明
idstring新建用户ID
namestring用户名
emailstring邮箱
成功响应示例
{ "id": "u_002", "name": "Bob", "email": "bob@example.com" }

GET/api/v2/categories获取分类列表
返回所有商品分类的树形结构
参数
参数名类型必填位置默认值说明
parent_idstring否query-父分类ID，为空则返回顶级分类
levelinteger否query1分类层级筛选
返回字段
字段类型说明
200成功返回分类列表
200.categoriesarray分类列表
成功响应示例
{ "categories": [ { "id": "cat_001", "name": "电子产品", "children": [ { "id": "cat_002", "name": "手机" }, { "id": "cat_003", "name": "笔记本电脑" } ] } ] }

POST/api/v2/categories创建分类
创建新的商品分类
参数
参数名类型必填位置默认值说明
namestring是body-分类名称
parent_idstring否body-父分类ID
iconstring否body-分类图标URL
sort_orderinteger否body-排序权重
返回字段
字段类型说明
201创建成功
201.idstring分类ID
201.namestring分类名称
成功响应示例
{ "id": "cat_010", "name": "智能穿戴" }

PUT/api/v2/categories/{id}更新分类
更新指定分类的信息
参数
参数名类型必填位置默认值说明
idstring是path-分类ID
namestring否body-分类名称
iconstring否body-图标URL
sort_orderinteger否body-排序权重
返回字段
字段类型说明
200更新成功
404分类不存在
成功响应示例
{ "id": "cat_010", "name": "智能穿戴设备", "icon": "https://example.com/icons/wearable.png", "sort_order": 5 }
错误响应示例
{ "error": "not_found", "message": "分类不存在" }

DELETE/api/v2/categories/{id}删除分类
删除指定分类（需无子分类和关联商品）
参数
参数名类型必填位置默认值说明
idstring是path-分类ID
返回字段
字段类型说明
204删除成功，无返回内容
400分类下存在子分类或商品，无法删除
错误响应示例
{ "error": "conflict", "message": "该分类下存在 3 个子分类，请先删除或迁移" }
