# 暗物质 WIMP 核反冲信号模拟

本程序用于模拟暗物质粒子（WIMP，大质量弱相互作用粒子）与探测器原子核反冲的信号，支持**年度调制效应**和**方向性探测**的各向异性信号模拟。

## 物理原理

### 1. 弹性散射理论
WIMP与靶核发生弹性散射时，反冲核能谱由以下微分截面公式描述：

```
dσ/dE_r = (σ₀ · A² · F²(Q)) / (2 · μ² · v²)
```

其中：
- `σ₀`：零动量转移截面
- `A`：靶核质量数
- `F(Q)`：核形状因子
- `μ`：WIMP与靶核的约化质量
- `v`：WIMP速度

### 2. 麦克斯韦-玻尔兹曼速度分布
程序使用**平滑截断**的麦克斯韦-玻尔兹曼分布描述WIMP的速度分布，并考虑地球相对于银河系暗物质晕的运动：

```
f(v) ∝ v · [exp(-(v-vₑ)²/v₀²) - exp(-(v+vₑ)²/v₀²)] · S(v)
```

其中：
- `v₀ = 220 km/s`：WIMP特征速度（银盘圆周速度）
- `vₑ = 232 km/s`：地球相对于暗物质晕的速度
- `v_esc = 544 km/s`：银河系逃逸速度
- `S(v)`：**平滑截断函数**（基于误差函数）

#### ✅ 重要改进：平滑截断 vs 硬截断

**问题**：原始的硬截断 `f(v) = 0 for v ≥ v_esc` 会导致：
- 速度分布在逃逸速度处不连续
- 能谱积分时出现伪尖峰（数值 artifacts）
- 事例生成时在能谱边缘出现异常结构

**解决方案**：使用基于误差函数的平滑截断：

```
S(v) = 0.5 · [1 + erf(5 · (0.5 - x))]

其中 x = (v - (v_esc - Δv)) / Δv，Δv = 0.05·v_esc
```

在过渡区域 `[v_esc-Δv, v_esc+Δv]` 内，分布平滑地从 1 降为 0，
确保了函数及其导数的连续性，避免了数值伪影。

### 3. 年度调制效应

由于地球绕太阳运动，地球相对于暗物质晕的速度随时间变化，导致反冲能谱出现**年度调制**：

```
v_earth(t) = √[v_LSR² + v_orb² + 2·v_LSR·v_orb·cos(γ)·cos(ωt - φ₀)]
```

其中：
- `v_LSR = 232 km/s`：本地静止标准速度
- `v_orb = 29.8 km/s`：地球轨道速度
- `γ = 60.2°`：黄道面与银道面夹角
- `ω = 2π/365.25 rad/day`：轨道角速度
- `φ₀`：相位常数（峰值约在6月2日）

### 4. 方向性探测（各向异性信号）

WIMP风具有明显的方向性（来自天鹅座方向），反冲核的角分布呈现**前向后向不对称**：

```
d²R/dE_r dΩ ∝ ∫(v_min→v_esc) f(v, θ) / v dv
```

其中 `θ` 是反冲核径迹与WIMP风方向的夹角。

**前向后向比 (F/B)**：
```
F/B = ∫(cosθ≥0) d²R/dE_r dΩ / ∫(cosθ<0) d²R/dE_r dΩ
```

典型 F/B 比值约为 2-5，取决于反冲能量和探测器类型。

### 5. 微分事例率
微分事例率（单位：事例/keV/kg/day）由下式计算：

```
dR/dE_r = (ρ₀ / (2 · m_χ · m_T)) · σ₀ · A² · F²(E_r) · ∫(v_min→v_esc) f(v)/v dv
```

其中：
- `ρ₀ = 0.3 GeV/c²/cm³`：本地暗物质密度
- `m_χ`：WIMP质量
- `m_T`：靶核质量

### 6. 核形状因子
使用Helm形式因子描述有限核大小对散射的影响：

```
F(q) = 3 · j₁(qR) / (qR) · exp(-(qs)²/2)
```

其中 `j₁` 是一阶球贝塞尔函数。

## 程序功能

### 主要类：WIMPSimulation

#### 初始化参数
| 参数 | 说明 | 默认值 |
|------|------|--------|
| `wimp_mass_gev` | WIMP质量 (GeV/c²) | 100 |
| `cross_section_cm2` | WIMP-质子散射截面 (cm²) | 1e-45 |
| `target_mass_amu` | 靶核质量 (amu) | 72.63 (锗) |
| `exposure_kg_day` | 探测器曝光量 (kg·day) | 1000 |
| `detector_type` | 探测器类型 | 'directional' |

#### 探测器类型
| 类型 | 名称 | 角分辨率 | 探测效率 |
|------|------|----------|----------|
| `emulsion` | 核乳胶探测器 | 15° | 85% |
| `gas` | 时间投影室(TPC) | 30° | 95% |
| `directional` | 通用方向性探测器 | 20° | 90% |

#### 主要方法

**基础能谱**：
1. `generate_spectrum(er_min, er_max, n_points)`：生成反冲能谱
2. `generate_events(n_events, er_min, er_max)`：生成模拟事例
3. `plot_spectrum()`：绘制微分能谱图
4. `plot_event_distribution()`：绘制事例能量分布
5. `print_parameters()`：打印模拟参数和预期事例数

**年度调制**：
6. `earth_velocity_day_of_year(day_of_year)`：计算某日地球速度
7. `differential_rate_time(er_kev, day_of_year)`：时间相关微分能谱
8. `annual_modulation_curve(er_min, er_max, n_days)`：生成年度调制曲线
9. `plot_annual_modulation()`：绘制年度调制曲线图

**方向性探测**：
10. `differential_rate_directional(er_kev, cos_theta)`：角度微分谱
11. `angular_distribution(er_kev, n_angles)`：角分布计算
12. `generate_directional_events(n_events, er_min, er_max)`：生成带方向的事例
13. `compute_forward_backward_ratio(er_min, er_max)`：计算前向后向比
14. `plot_angular_distribution()`：绘制角分布图（含极坐标）
15. `plot_directional_events()`：绘制方向性探测模拟图
16. `compare_detector_types()`：对比不同探测器类型

## 使用方法

### 环境要求
- Python 3.7+
- NumPy
- Matplotlib
- SciPy

### 安装依赖
```bash
pip install numpy matplotlib scipy
```

### 基本使用

```python
from wimp_recoil_simulation import (
    WIMPSimulation, 
    compare_masses,
    validate_smooth_cutoff,
    compare_spectra_smoothness
)

# 验证平滑截断效果（推荐首次运行）
validate_smooth_cutoff()
compare_spectra_smoothness()

# 创建模拟实例（核乳胶探测器）
sim = WIMPSimulation(
    wimp_mass_gev=50,           # 50 GeV WIMP
    cross_section_cm2=1e-45,    # 1e-45 cm² 截面
    target_mass_amu=72.63,      # 锗靶
    exposure_kg_day=1000,       # 1000 kg·day 曝光量
    detector_type='emulsion'    # 核乳胶探测器
)

# 打印参数和预期结果
sim.print_parameters()

# 生成并绘制基础能谱
sim.plot_spectrum(er_min=1, er_max=100)

# 生成模拟事例并绘制分布
sim.plot_event_distribution(n_events=5000)

# 年度调制分析
sim.plot_annual_modulation(er_min=2, er_max=6)

# 方向性分析
sim.plot_angular_distribution(er_kev=20)
sim.plot_directional_events(n_events=2000)
sim.compare_detector_types(er_kev=20)
```

### 运行主程序
```bash
python wimp_recoil_simulation.py
```
