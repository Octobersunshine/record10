# 伊辛模型模拟套件 (Go语言实现)

## 项目概述

本项目实现了三种伊辛模型模拟算法，涵盖纯净系统和无序系统：

1. **纯净伊辛模型** + Swendsen-Wang簇算法
2. **随机横场伊辛模型(RTFIM)** + 平行回火(Parallel Tempering)
3. **自旋玻璃相图计算**

## 快速开始

### 编译运行
```bash
go mod init ising
go build -o ising.exe
```

### 运行模式

#### 1. 纯净伊辛模型 (默认)
```bash
ising.exe
# 或
ising.exe -mode=pure
```

#### 2. 自旋玻璃单组参数
```bash
ising.exe -mode=glass -h=1.0 -hw=0.5 -n=5
```
参数说明：
- `-h`: 平均横场强度
- `-hw`: 横场无序宽度 (0=均匀场)
- `-n`: 无序样本数 (越多统计越准)

#### 3. 自旋玻璃相图计算
```bash
ising.exe -mode=diagram -n=3
```

## 第一部分：纯净伊辛模型

### 算法：Swendsen-Wang簇算法

| 特性 | Metropolis | Swendsen-Wang |
|------|-----------|---------------|
| 更新方式 | 单自旋翻转 | 多自旋聚类翻转 |
| 动态临界指数 | z ≈ 2.17 | z ≈ 0.2 |
| 热化步数 | 10000+ | 500 |

### 物理量
- ⟨E⟩: 平均能量
- ⟨|M|⟩: 平均磁化强度
- Cv: 比热 (能量涨落)
- χ: 磁化率 (磁化涨落)
- U₄: Binder累积量 (有限尺寸标度)

**临界温度**: Tc ≈ 2.269 (Onsager精确解)

## 第二部分：随机横场伊辛模型 (RTFIM)

### 哈密顿量

```
H = -J Σ <i,j> σ_i^z σ_j^z - Σ h_i σ_i^x
```

- σ_i^z = cosθ_i (纵向自旋分量)
- σ_i^x = sinθ_i (横向自旋分量)
- h_i: 随机横场 (均匀分布: [hMean-hWidth, hMean+hWidth])

### 序参量

| 序参量 | 公式 | 物理意义 |
|--------|------|----------|
| 磁化强度 | ⟨|M_z|⟩ = ⟨|Σ σ_i^z|⟩ | 铁磁有序 |
| Edwards-Anderson | q_EA = ⟨σ_i^z⟩² 构型平均 | 自旋玻璃冻结 |
| 重叠函数 | q = (1/N) Σ ⟨σ_i^a σ_i^b⟩ | 两副本相似性 |

## 第三部分：平行回火算法 (Parallel Tempering)

### 核心思想

自旋玻璃的能量地形粗糙多谷，传统蒙特卡洛容易陷入局域极小。平行回火通过**多副本交换**跨越能垒：

1. 同时运行 M 个副本，温度: T₁ < T₂ < ... < T_M
2. 每个副本独立进行蒙特卡洛更新
3. 定期尝试交换相邻温度的副本配置：

   接受概率: P = min[1, exp((β_i - β_j)(E_i - E_j))]

### 算法优势

| 问题 | 解决方案 |
|------|----------|
| 能量谷之间能垒高 | 高温副本容易跨越 |
| 低温配置难平衡 | 从高温侧"注入"平衡态 |
| 临界慢化 | 副本交换加速全局混合 |

### 本实现参数

- 副本数: 12 (温度范围 0.2 - 4.0)
- 交换频率: 每10次扫描尝试一次交换
- 无序平均: 多个随机场样本取平均

## 第四部分：自旋玻璃相图

### 相区说明

```
          h (横场)
           ↑
           |  量子顺磁态 (QPM)
           |    M_z=0, q_EA=0
     h_c   |..........
           |         \
           |          \  自旋玻璃相 (SG)
           |           \  M_z=0, q_EA>0
           |            \
           |             \
           |              \  铁磁相 (FM)
           |               \  M_z>0, q_EA>0
           --------------------------------→ T (温度)
                          T_c
```

### 相图特征

1. **T=0 量子相变**：h 从 0 增加到 h_c 时，发生铁磁→顺磁量子相变
2. **经典相变**：h=0 时，T_c≈2.27 对应纯净伊辛模型
3. **玻璃区**：中等 h 和 T 时出现自旋玻璃相

## 技术细节

### Union-Find 数据结构
- 路径压缩 (Path Compression)
- 按大小合并 (Union by Size)
- 复杂度: 接近 O(1) 每次操作

### 无序平均
对每个随机场实现独立模拟，物理量取多个无序样本的平均值：
```
⟨O⟩_disorder = (1/N_s) Σ ⟨O⟩_thermal[α]
```
其中 α 标记不同的随机场构型。
