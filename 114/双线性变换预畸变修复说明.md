# 双线性变换频率预畸变修复说明

## 问题背景

双线性变换是将模拟滤波器转换为数字滤波器的标准方法。然而，双线性变换在频率轴上引入了非线性压缩，导致截止频率偏差。

### 双线性变换公式:
```
s = (2/T) * (1 - z^-1) / (1 + z^-1)
```

### 频率映射关系:
```
Ω = (2/T) * tan(ω/2)
```

其中:
- `Ω` - 模拟角频率 (rad/s)
- `ω` - 数字角频率 (rad/sample)
- `T = 1/fs` - 采样周期

## 问题表现

如果不进行预畸变：
1. 设计的数字滤波器时，关键频率点（通带截止、阻带截止）会偏离目标值
2. 频率越高，偏差越大
3. 接近奈奎斯特频率 (fs/2) 时偏差最严重

## 修复方案: 频率预畸变

### 预畸变公式

```
Ω_a = (2/T) * tan(ω_d * T / 2)
```

简化为频率 (Hz):
```
f_a = (fs/π) * tan(π * f_d / fs)
```

### 设计流程 (修正后)

```
数字频率 (目标)
     ↓
[频率预畸变)
     ↓
模拟频率 (预畸变后)
     ↓
设计模拟滤波器
     ↓
双线性变换
     ↓
数字滤波器 (频率准确)
```

## 代码实现

### BilinearTransformer 类 (`iir_filter_fixed.py:6-29`)

```python
class BilinearTransformer:
    @staticmethod
    def prewarp(freq_hz, fs):
        T = 1.0 / fs
        omega_d = 2 * np.pi * freq_hz
        omega_a = (2.0 / T) * np.tan(omega_d * T / 2.0)
        return omega_a / (2 * np.pi)
```

### IIRFilterDesigner 类中的应用

```python
def butterworth(self, filter_type, passband_freq, stopband_freq, ..., prewarp=True):
    if prewarp:
        # 步骤1: 频率预畸变
        wp, ws = self._prewarp_frequencies(passband_freq, stopband_freq)
        
        # 步骤2: 设计模拟滤波器
        b, a, N = self._analog_butterworth(btype, wp, ws, ...)
        
        # 步骤3: 双线性变换
        b, a = self._bilinear_transform(b, a)
    else:
        # 无预畸变 (直接数字设计)
        ...
```

## 使用方法

### 基础使用 (推荐: 带预畸变 (默认)

```python
from iir_filter_fixed import IIRFilterDesigner

designer = IIRFilterDesigner(fs=1000.0, verbose=True)

# 带预畸变的滤波器设计
filter_obj = designer.butterworth(
    filter_type='lowpass',
    passband_freq=100.0,      # 目标通带截止频率
    stopband_freq=150.0,      # 目标阻带截止频率
    passband_ripple=1.0,
    stopband_attenuation=40.0,
    prewarp=True  # 默认开启
)
```

### 关闭预畸变 (用于对比)

```python
filter_obj = designer.butterworth(
    ...,
    prewarp=False
)
```

## 关键结论

1. **双线性变换固有的频率非线性**是不可避免的
2. **频率预畸变**是标准的补偿方法
3. **预畸变确保关键频率点准确映射
4. **scipy 内部也做预畸变，但显式实现更透明
