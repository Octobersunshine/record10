# IIR 数字滤波器设计工具使用说明

## 环境要求

```bash
pip install numpy scipy
```

## 快速开始

### 基本使用示例

```python
from iir_filter_enhanced import IIRFilterDesigner, print_filter_info

# 1. 创建设计器 (采样频率 1000 Hz)
designer = IIRFilterDesigner(fs=1000.0)

# 2. 设计滤波器
filter_obj = designer.butterworth(
    filter_type='lowpass',           # 滤波器类型: lowpass, highpass, bandpass, bandstop
    passband_freq=100.0,             # 通带截止频率 (Hz)
    stopband_freq=150.0,             # 阻带截止频率 (Hz)
    passband_ripple=1.0,             # 通带最大波纹 (dB)
    stopband_attenuation=40.0,       # 阻带最小衰减 (dB)
    return_sos=True                  # 返回二阶节形式 (推荐用于实时处理)
)

# 3. 查看滤波器信息
print_filter_info(filter_obj)

# 4. 实时处理信号
input_sample = 1.23  # 输入一个样本
output_sample = filter_obj.process_sample(input_sample)  # 处理样本

# 5. 导出 C 语言头文件
filter_obj.export_c_header('filter_coeffs.h')
```

## 滤波器类型

### 1. 巴特沃斯 (Butterworth)

```python
filter_obj = designer.butterworth(
    filter_type='lowpass',
    passband_freq=100.0,
    stopband_freq=150.0,
    passband_ripple=1.0,
    stopband_attenuation=40.0,
    return_sos=True
)
```

**特点:**
- ✅ 通带和阻带都平坦（最大平坦）
- ✅ 相位响应较好
- ❌ 需要较高阶数达到相同衰减
- **适用:** 通用滤波、音频、ECG

### 2. 切比雪夫 I 型 (Chebyshev Type I)

```python
filter_obj = designer.chebyshev1(
    filter_type='lowpass',
    passband_freq=100.0,
    stopband_freq=150.0,
    passband_ripple=1.0,
    stopband_attenuation=40.0,
    return_sos=True
)
```

**特点:**
- ✅ 通带有波纹，阻带单调
- ✅ 较低阶数达到高衰减
- ❌ 相位响应较差
- **适用:** 对相位要求不高，需要陡过渡带

### 3. 切比雪夫 II 型 (Chebyshev Type II)

```python
filter_obj = designer.chebyshev2(
    filter_type='lowpass',
    passband_freq=100.0,
    stopband_freq=150.0,
    passband_ripple=1.0,
    stopband_attenuation=40.0,
    return_sos=True
)
```

**特点:**
- ✅ 通带平坦，阻带有波纹
- ✅ 较低阶数达到高衰减
- **适用:** 陷波滤波器、希望通带平坦

### 4. 椭圆滤波器 (Elliptic)

```python
filter_obj = designer.elliptic(
    filter_type='lowpass',
    passband_freq=100.0,
    stopband_freq=150.0,
    passband_ripple=1.0,
    stopband_attenuation=40.0,
    return_sos=True
)
```

**特点:**
- ✅ 通带和阻带都有波纹
- ✅ 阶数最低，过渡带最陡
- ❌ 相位响应最差
- **适用:** 资源受限，对相位要求不高

## 滤波器类型参数

| 类型 | passband_freq | stopband_freq | 说明 |
|------|--------------|--------------|------|
| lowpass | 标量 (如 100.0) | 标量 (如 150.0) | 低通 |
| highpass | 标量 (如 100.0) | 标量 (如 50.0) | 高通 |
| bandpass | 元组 (如 (80, 120)) | 元组 (如 (60, 140)) | 带通 |
| bandstop | 元组 (如 (45, 55)) | 元组 (如 (49, 51)) | 带阻/陷波 |

### C 语言实现 (二阶节)

```c
// 二阶节滤波器结构
typedef struct {
    float b0, b1, b2;  // 分子系数
    float a1, a2;      // 分母系数 (a0 = 1)
    float x1, x2;      // 输入状态
    float y1, y2;      // 输出状态
} Biquad;

// 二阶节处理函数
float biquad_process(Biquad *f, float x) {
    float y = f->b0 * x + f->b1 * f->x1 + f->b2 * f->x2
                        - f->a1 * f->y1 - f->a2 * f->y2;
    f->x2 = f->x1;
    f->x1 = x;
    f->y2 = f->y1;
    f->y1 = y;
    return y;
}

// 多阶 SOS 滤波器处理
#define NUM_SECTIONS 4
Biquad sections[NUM_SECTIONS];

float iir_process(float x) {
    for (int i = 0; i < NUM_SECTIONS; i++) {
        x = biquad_process(&sections[i], x);
    }
    return x;
}
```

## 注意事项

1. **优先使用二阶节 (SOS):** 高阶滤波器直接形式容易出现数值不稳定问题
2. **系数量化:** 嵌入式实现时注意系数量化精度
3. **动态范围:** 定点数实现时注意中间结果的动态范围
4. **群延迟:** IIR 滤波器有非线性相位，注意信号失真
5. **稳定性:** 设计完成后检查极点是否在单位圆内
