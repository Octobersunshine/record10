# 黎曼Zeta函数非平凡零点计算工具 (高精度版)

## 简介

本工具使用Python的mpmath库计算黎曼Zeta函数的非平凡零点，并提供统计分析功能用于随机矩阵理论对照。实现了完整的黎曼-西格尔公式，包含自适应截断和容差控制，确保零点定位精度达到1e-12以上。

## 环境要求

- Python 3.7+
- mpmath 库

## 安装依赖

```bash
pip install mpmath
```

### 核心脚本

1. **riemann_zeta_zeros.py** - 高精度零点计算主程序

   - 自适应截断和容差控制 (默认1e-12)
   - 二分法+牛顿法混合零点精化
   - 使用mpmath内置zeta函数双重验证

## 算法改进说明

### 1. Theta函数高阶修正

原版本仅使用2项修正，新版本扩展到9项：
```
θ(t) = (t/2)ln(t/2π) - t/2 - π/8 
     + 1/(48t) + 7/(5760t³) + 31/(80640t⁵) + ...
```

### 2. 自适应截断控制

主求和项根据容差动态截断，当连续两项都小于 `tol/(100*N)` 时停止求和。

### 3. 余项计算

实现了黎曼-西格尔公式的余项计算，进一步提高精度。

### 4. 混合寻根算法

- 首先使用自适应步长搜索符号变化
- 发现符号变化后使用细步精确定位区间
- 使用二分法（稳定）和牛顿法（快速）混合精化
- 最终用 `|ζ(0.5+it)| < 1e-8` 验证零点有效性

## 数学原理

### 黎曼-西格尔公式

Z(t) = 2 * Σ(n=1 to N) cos(θ(t) - t ln n) / √n + R(t)

其中:
- N = floor(√(t/(2π)))
- R(t) 是余项
- θ(t) 是黎曼-西格尔theta函数

### Xi函数方法

ξ(s) = ζ(s) · Γ(s/2) · π^(-s/2) · (s-1)/2

ξ(1/2 + it) 是实值函数，其零点对应Zeta函数的非平凡零点。

### 零点计数函数 N(T)

N(T) = (T/2π) log(T/2π) - T/2π + 7/8 + S(T)

其中 S(T) = (1/π) arg ζ(1/2 + iT) 是零点计数的修正项。

### 归一化间隔

δ_n = (γ_{n+1} - γ_n) · (1/2π) log(γ_n/2π)

归一化后的间隔平均值应为1，便于与随机矩阵理论对比。

### 随机矩阵理论 (GUE Wigner推测)

GUE (Gaussian Unitary Ensemble) 的最近邻间距分布:

p_Wigner(s) = (π/2) s exp(-π s²/4)

F_Wigner(s) = 1 - exp(-π s²/4)

### Poisson分布 (独立随机变量)

p_Poisson(s) = exp(-s)

F_Poisson(s) = 1 - exp(-s)

**关键区别**: GUE分布在s→0时p(s)→0 (能级排斥)，而Poisson分布p(0)=1。
