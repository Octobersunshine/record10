# Go 数值求导程序

支持**前向模式自动微分**和**中心差分数值微分**的高精度求导工具。

## 功能特性

- 支持自定义函数表达式
- **两种微分方法**：
  - **自动微分（前向模式）**：基于双数（Dual Number）的精确符号级求导
  - **中心差分（数值微分）**：传统数值逼近方法
- 支持指定求导点 x0
- 支持自定义步长 h（仅数值微分）
- 内置常用数学函数支持（sin, cos, tan, exp, log, sqrt, abs）
- 不连续点检测：自动检测不可导点并发出警告
- 自动选择最优微分方法

## 自动微分原理（前向模式）

使用**双数（Dual Number）**：`a + bε`，其中 `ε² = 0`

对于函数 `f(x)`，将自变量 `x` 表示为双数 `x0 + 1·ε`，则：
- `f(x0 + ε) = f(x0) + f'(x0)·ε`
- 结果的实部是函数值，虚部就是导数值

### 支持的运算规则

| 运算 | 函数值 | 导数值 |
|------|--------|--------|
| `u + v` | `u.Val + v.Val` | `u.D + v.D` |
| `u - v` | `u.Val - v.Val` | `u.D - v.D` |
| `u * v` | `u.Val * v.Val` | `u.D*v.Val + u.Val*v.D` |
| `u / v` | `u.Val / v.Val` | `(u.D*v.Val - u.Val*v.D) / v.Val²` |

### 支持的函数规则

| 函数 | 导数规则 |
|------|----------|
| `sin(u)` | `cos(u) * u'` |
| `cos(u)` | `-sin(u) * u'` |
| `tan(u)` | `sec²(u) * u' = 1/cos²(u) * u'` |
| `exp(u)` | `exp(u) * u'` |
| `log(u)` | `(1/u) * u'` |
| `sqrt(u)` | `(1/(2*sqrt(u))) * u'` |
| `abs(u)` | `sign(u) * u'`（u=0时为NaN） |

## 编译运行

```bash
go build -o derivative.exe
```

## 使用方法

```bash
derivative.exe -f="函数表达式" -x=求导点 [-h=步长] [-method=方法]
```

### 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `-f` | `"x*x"` | 函数表达式 |
| `-x` | `0` | 求导点 x0 |
| `-h` | `0.001` | 步长（仅用于数值微分） |
| `-method` | `"auto"` | 微分方法: `auto`-自动选择, `num`-数值微分, `ad`-自动微分 |

### 支持的运算

- 基本运算：+, -, *, /
- 括号：( )
- 数学函数：sin, cos, tan, exp, log, sqrt, abs
- 常量：pi, e

## 使用示例

### 示例1：多项式求导 f(x) = x³ + 2x² + 3x

```bash
derivative.exe -f="x*x*x + 2*x*x + 3*x" -x=2
```

**解析解**：f'(x) = 3x² + 4x + 3, f'(2) = 12 + 8 + 3 = 23

**输出**：
```
函数表达式: f(x) = x*x*x + 2*x*x + 3*x
求导点 x0 = 2.000000
函数值 f(2) = 26.0000000000
使用方法: 自动微分（前向模式）
一阶导数 f'(2) = 23.000000000000000
```

### 示例2：复合函数 f(x) = sin(x²) + exp(-x)

```bash
derivative.exe -f="sin(x*x) + exp(-x)" -x=1
```

**解析解**：f'(x) = 2x·cos(x²) - exp(-x), f'(1) ≈ 2*cos(1) - 1/e ≈ 0.7115

**输出**：
```
函数表达式: f(x) = sin(x*x) + exp(-x)
求导点 x0 = 1.000000
函数值 f(1) = 1.2087793479
使用方法: 自动微分（前向模式）
一阶导数 f'(1) = 0.711510855998939
```

### 示例3：对比两种方法的精度

**自动微分（精确）**：
```bash
derivative.exe -f="exp(sin(x))" -x=0.5 -method=ad
```

**数值微分（近似）**：
```bash
derivative.exe -f="exp(sin(x))" -x=0.5 -method=num -h=0.001
```

**解析解**：f'(x) = exp(sin(x)) · cos(x), f'(0.5) ≈ 1.7165

### 示例4：不连续点检测 - f(x) = |x|

```bash
derivative.exe -f="abs(x)" -x=0
```

**输出**：
```
函数表达式: f(x) = abs(x)
求导点 x0 = 0.000000
警告: 自动微分检测到该点不可导（如 abs 在 x=0 处）
函数值 f(0) = 0.0000000000
使用方法: 自动微分（前向模式）
一阶导数 f'(0) = NaN
```

## 方法对比

| 特性 | 自动微分（前向模式） | 中心差分（数值微分） |
|------|---------------------|---------------------|
| 精度 | 机器精度（精确） | 依赖步长（近似） |
| 步长 | 不需要 | 需要选择合适步长 |
| 不连续点 | 返回NaN明确标识 | 返回不可靠近似值 |
| 复杂表达式 | 完全支持，精度不变 | 累积误差 |
| 性能 | 单次表达式求值 | 2次表达式求值 |
| 适用场景 | 大多数光滑函数 | 自动微分不可用时的后备 |

## 自动选择策略（method=auto）

程序会先进行连续性检测：
1. 如果函数在该点连续可导 → 使用**自动微分**（高精度）
2. 如果检测到不连续或不可导 → 使用**数值微分**（并发出警告）
