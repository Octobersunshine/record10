# 有限元模态分析工具使用说明

## 概述

本工具用于求解线性自由振动的广义特征值问题：

$$ K\Phi = \omega^2 M\Phi $$

其中：
- $K$ - 刚度矩阵
- $M$ - 质量矩阵
- $\omega$ - 固有角频率 (rad/s)
- $\Phi$ - 振型向量

## 环境要求

- Python 3.7+
- NumPy
- SciPy

安装依赖：
```bash
pip install numpy scipy
```

## 文件说明

### 1. modal_analysis.py - 基础版
- 函数式编程风格
- 包含两个示例：弹簧质量系统、简支梁模型
- 适合快速使用

### 2. modal_analysis_viz.py - 增强版
- 面向对象编程风格
- 额外功能：
  - 模态参与因子计算
  - 模态质量/刚度计算
  - 三个完整示例

## 核心功能

### 特征值求解
- 使用 `scipy.linalg.eigh` 求解广义特征值问题
- 自动按频率从小到大排序
- 支持指定求解模态数（适用于大型系统）

### 振型归一化
1. **质量归一化** (`normalize='mass'`)：
   $$ \phi_i^T M \phi_i = 1 $$

2. **最大值归一化** (`normalize='max'`)：
   - 振型最大绝对值归一化为1

3. **不归一化** (`normalize=None`)

### 结果验证
- 振型正交性验证
- 质量矩阵正交性：$\Phi^T M \Phi$ 应为对角阵
- 刚度矩阵正交性：$\Phi^T K \Phi$ 应为对角阵

## 使用示例

### 基础版使用方法

```python
import numpy as np
from modal_analysis import modal_analysis, print_modal_results, verify_orthogonality

# 定义刚度矩阵和质量矩阵
K = np.array([
    [2000, -1000],
    [-1000, 2000]
])
M = np.diag([1.0, 1.0])

# 求解
frequencies, natural_frequencies, mode_shapes = modal_analysis(
    K, M, 
    num_modes=2,      # 求解前2阶模态
    normalize='mass'  # 质量归一化
)

# 打印结果
print_modal_results(frequencies, natural_frequencies, mode_shapes)

# 验证正交性
verify_orthogonality(K, M, mode_shapes, frequencies)
```

### 增强版使用方法

```python
import numpy as np
from modal_analysis_viz import ModalAnalysis

# 创建分析对象
K = np.array([[2000, -1000], [-1000, 2000]])
M = np.diag([1.0, 1.0])
ma = ModalAnalysis(K, M)

# 求解
ma.solve(num_modes=2, normalize='mass')

# 获取结果
frequencies = ma.frequencies           # 固有频率 (Hz)
natural_freqs = ma.natural_frequencies # 角频率 (rad/s)
mode_shapes = ma.mode_shapes           # 振型矩阵

# 辅助计算
participation = ma.get_participation_factors()  # 模态参与因子
modal_mass = ma.get_modal_mass()                # 模态质量
modal_stiff = ma.get_modal_stiffness()          # 模态刚度

# 打印和验证
ma.print_results()
ma.verify_orthogonality()
```

## 注意事项

1. **矩阵维度**：K和M必须是同维方阵
2. **刚体模态**：如果存在刚体模态（零频率），需确保约束正确
3. **数值稳定性**：对于病态矩阵，建议进行预处理
4. **大型系统**：使用 `num_modes` 参数只求解前几阶模态，提高效率

## 理论背景

### 自由振动方程
无阻尼自由振动的运动方程：
$$ M\ddot{u} + Ku = 0 $$

假设简谐解：
$$ u(t) = \phi e^{i\omega t} $$

代入得到广义特征值问题：
$$ (K - \omega^2 M)\phi = 0 $$

### 振型正交性
对于不同阶的振型，满足：
$$ \phi_i^T M \phi_j = 0 \quad (i \neq j) $$
$$ \phi_i^T K \phi_j = 0 \quad (i \neq j) $$

这一性质是模态叠加法的基础。
