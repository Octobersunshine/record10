# 打靶法求解地震射线路径和走时 - 已修复折射盲区

## 简介

本项目使用 Python 实现了打靶法（Shooting Method）来求解地震波在不同速度模型中的射线路径和走时。
**已修复层状模型中的折射盲区问题**，能够正确处理临界角入射时的全反射和滑行波现象。

## 功能特性

- 支持两种速度模型：
  - **梯度速度模型**：速度随深度线性增加 v(z) = v0 + k*z
  - **层状速度模型**：多层介质，每层速度恒定
- **事件驱动射线追踪**：自动检测层界面并处理折射/反射
- **斯涅尔定律折射计算**：精确计算折射角
- **临界角检测**：识别并正确处理全反射
- **混合优化策略**：全局搜索 + 局部优化，解决折射盲区收敛问题
- 使用龙格-库塔法（RK45）求解射线方程
- 可视化射线路径和速度模型

## 折射盲区修复说明

### 问题分析

在层状速度模型中，当射线从低速层入射到高速层时：
- **出射角 < 临界角 (θc)**：发生正常折射
- **出射角 ≈ 临界角**：产生沿界面滑行的折射波，形成折射盲区，常规打靶法不收敛
- **出射角 > 临界角**：发生全反射

### 修复方案

1. **事件驱动射线追踪**：使用 solve_ivp 的 events 参数精确检测射线何时穿越层界面
2. **斯涅尔定律实现**：在层界面处根据入射角和速度比计算折射角
3. **临界角检测**：θc = arcsin(v1/v2)，当入射角大于等于临界角时执行全反射
4. **混合优化策略**：
   - 首先在角度范围内进行 50 点全局粗搜索，找到最佳初始角度
   - 然后使用 minimize_scalar 进行局部精细优化
   - 使用深度匹配目标函数，确保射线在目标深度附近被拾取

## 使用方法

### 1. 运行示例

```bash
python shooting_method.py
```

程序将自动演示：
- 临界角与折射盲区分析
- 梯度速度模型示例
- 层状速度模型示例（已修复折射盲区）

### 2. 自定义使用

```python
from shooting_method import VelocityModel, ShootingMethod, plot_rays

# 创建速度模型
vm = VelocityModel(
    model_type='gradient',
    params={'v0': 2000, 'k': 0.5}  # v0: 表面速度, k: 梯度
)

# 或层状模型
vm = VelocityModel(
    model_type='layered',
    params={
        'layers': [
            {'z_top': 0, 'z_bottom': 1000, 'v': 2000},
            {'z_top': 1000, 'z_bottom': 2000, 'v': 3000},
        ]
    }
)

# 创建打靶法求解器（开启debug可查看详细折射/反射信息）
sm = ShootingMethod(vm, debug=True)

# 定义震源和接收点
source = (0, 0)  # (x, z)
receivers = [(2000, 1000), (4000, 1500)]

# 计算所有射线路径
rays = sm.compute_rays(source, receivers)

# 绘制结果
plot_rays(rays, source, vm, xlim=(-1000, 5000), zlim=(2000, -100), title="My Model")
```

## 算法原理

### 射线方程

基于程函方程的射线追踪系统：
- dx/ds = p * v
- dz/ds = q * v
- dp/ds = 0
- dq/ds = -∇v / v²

其中：
- (p, q) 为慢度矢量
- v 为速度
- s 为射线路径长度

### 斯涅尔定律折射

在层界面处，根据斯涅尔定律：
- sin(θ1) / v1 = sin(θ2) / v2
- 临界角 θc = arcsin(v1/v2)（当 v2 > v1 时存在）

### 打靶法

1. 在角度范围内进行全局粗搜索（50个采样点）
2. 选择误差最小的角度作为初始值
3. 使用 Brent 方法进行局部精细优化
4. 使用深度匹配的目标函数确保收敛
